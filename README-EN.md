build:
```
git clone https://github.com/umzi2/avif2png-rs
cargo build --release
```

linux:
```
chmod +x avif2png-rs 
./avif2png-rs -i <путь> -o <путь>
```
windows:
```
avif2png-rs.exe -i <путь> -o <путь>
```
The repository was taken as a basis kornelski: [avif-decode](https://github.com/kornelski/avif-decode)
